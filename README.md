# Month Names Script

## Description
A basic PHP script to display month names in Turkish.

## Features
- Lists all months in Turkish.

## Installation
No installation required. Simply run the script on a server with PHP support.

## Usage
```php
$aylar = array("Ocak","Şubat","Mart","Nisan","Mayıs","Haziran","Temmuz",
  "Ağustos","Eylül","Ekim","Kasım","Aralık");

foreach($aylar as $değer){
echo "$değer   ";
}
```
